# TuxDiscordBot
Tux - A Discord Bot for some basic server role operations

## To run the bot
replace "BOT TOKEN" with your Discord bot token you get from the Discord dev platform  
then run ```python main.py``` to start the bot

## Commands
admin should run ```!promote``` to promote "Class 9, Class 10, Class 11, Class 12" to respective increments ("Class 12" -> "Passout" and removes their "Active Member" role as well)  

running ```!active``` grants "Active Member" role to all those having "Class 9, Class 10, Class 11, Class 12" roles (this was needed to setup this system for the first time).
